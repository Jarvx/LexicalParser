本程序实现词法分析


对于给定输入正则文法和源代码 输出最小DFA并通过DFA对源程序进行
词法分析，输出token序列。

源文件说明：

DFA.cpp  用于给定正则文法，处理生成DFA

LexicalParser.cpp  用于给定DFA，进行词法分析，输出token序列

输入说明：

输入部分为两个文件 
grammer.txt 为给定正则文法
source_char 为给定输入源程序

输出说明：

mindfa.txt  为输出dfa，其中第一行为dfa状态数，第二行对应各个状态
之间边的数目。此后若干行为各条边起始状态编号、终止状态编号、边上
的条件字符。注：边上的条件用字符的ascii值 


source_token.txt 为输出的token序列

源代码中重要类、函数、变量均有注释。

程序不但向文件中写入结果，也将向屏幕输出相关处理信息。
